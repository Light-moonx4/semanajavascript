Aplicación de Notas en Vanilla JavaScriptUna aplicación web interactiva y ligera para gestionar notas en tiempo real. Este proyecto implementa la manipulación dinámica del DOM (Document Object Model) y persistencia de datos local utilizando JavaScript puro.🚀 
Características PrincipalesPersistencia Total: Las notas se guardan en el navegador y no se borran al recargar la página.Seguridad: Inserción segura de texto para prevenir ataques de inyección de código (XSS).Interactividad: Flujo completo de agregar, listar y eliminar elementos de forma dinámica.
🛠️ Tecnologías UtilizadasHTML5: Estructura semántica de la interfaz.CSS3: Estilos y diseño visual (clases de botones).JavaScript (ES6+): Lógica de programación y manejo del DOM.
📋 Arquitectura y Solución de TareasEl código está organizado estratégicamente para cumplir con los siguientes requerimientos técnicos:🔍 Selección de Elementos (Task 2)Se utilizan métodos óptimos y variados para capturar los nodos del DOM:getElementById para capturar el contenedor de la lista y el campo de texto de forma directa.
querySelector para seleccionar el botón principal a través de su clase CSS .btn-principal.
➕ Creación y Validación (Task 3)Validación: El sistema limpia los espacios en blanco (.trim()) y bloquea registros vacíos mediante alertas al usuario.Creación Segura: Se generan nodos <li> y <button> dinámicamente con document.createElement. Se usa .textContent para asignar texto de forma segura.
🗑️ Eliminación Eficiente (Task 4)Cada nota incluye su propio botón de eliminación con un escuchador de eventos (addEventListener).Al hacer clic, se remueve el elemento del arreglo mediante su índice y se borra visualmente del árbol HTML usando removeChild desde el nodo padre (<ul>).
💾 Persistencia de Datos (Task 5)Lectura: Al cargar la página, se extrae el string de localStorage y se transforma en un arreglo funcional mediante JSON.parse.Escritura: Cada acción de agregar o eliminar actualiza el almacenamiento local convirtiendo el arreglo a texto con JSON.stringify.
